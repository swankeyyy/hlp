Подключаем Offcanvas для меню с помощью Bootstrap.

Offcanvas — это компонент боковой панели, который можно переключать с помощью JavaScript, чтобы он появлялся с левого, правого, верхнего или нижнего края области просмотра. Кнопки или привязки используются в качестве триггеров, прикрепленных к конкретным элементам, которые вы переключаете, а dataатрибуты используются для вызова нашего JavaScript.
